> No meu TCC, criei um protótipo de sistema chamado 4Casting. Se tratava
de uma plataforma para conectar artistas e contratantes da região
centro-oeste de Minas Gerais, criando oportunidades de emprego para a
área artística no interior do estado. Aprovado com 90% da pontuação.

- [x] PHP
- [x] DomPDF
